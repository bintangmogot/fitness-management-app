# 🗺️ Sitemap
## Website Coach Yohan - MVP (Level 1)

---

## Overview

Website MVP adalah **Single-Page Landing Page** dengan beberapa section yang dapat di-navigate menggunakan anchor links.

---

## Page Structure

```
index.html (Single Page)
│
├── #hero ────────────── Hero Section
├── #about ───────────── About / Portfolio
├── #testimonials ────── Video Testimonials
├── #packages ────────── Pricing Packages  
├── #location ────────── Location & Gym Info
└── #contact ─────────── CTA / Contact
```

---

## Navigation Menu

| Menu Label | Anchor | Section |
|------------|--------|---------|
| Home | `#hero` | Hero dengan tagline |
| About | `#about` | Profile & portfolio |
| Testimonials | `#testimonials` | Video carousel |
| Packages | `#packages` | Daftar harga |
| Location | `#location` | Maps & gym info |
| Contact | `#contact` | WhatsApp CTA |
